# Analytics App

Track and analyze user activity and course performance.

## Models

- **CourseAnalytics** - Course-level analytics
- **UserActivity** - User action tracking
- **StudentAnalytics** - Student performance analytics

## API Endpoints

- `GET /api/analytics/courses/{id}/` - Course analytics
- `GET /api/analytics/users/{id}/` - User analytics
- `GET /api/analytics/dashboard/` - Admin dashboard data

## Features

- User activity tracking
- Course engagement metrics
- Student performance analysis
- Completion rates
- Time spent tracking
